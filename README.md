# 键盘练习游戏

一个现代化的键盘指法练习游戏，支持分阶段训练、指法分析和个性化学习。

## 功能特点

### 🎮 四种难度模式

| 模式 | 描述 | 特色 |
|------|------|------|
| **初级模式** | 分10阶段循序渐进的指法训练 | 基准键位→各列纵向练习→综合测试 |
| **中级模式** | 英文短句/短文打字练习 | 实时准确率、CPM/WPM 统计 |
| **高级模式** | 字母下落切水果玩法 | 指法均衡算法，确保每根手指均衡练习 |
| **大师模式** | 汉字拼音输入挑战 | 拼音打字 + 汉字学习 |

### ✨ 核心功能

- **分阶段训练系统**：10个递进学习阶段，从基准键位逐步扩展到全键盘
- **指法合规追踪**：记录并分析每次按键的手指使用是否正确
- **指法均衡算法**：智能分配字母，确保十指均衡发展
- **姿势指导模块**：内置打字姿势教程，帮助养成良好习惯
- **全程音效动画**：即时视觉和音效反馈
- **鼓励系统**：积极反馈和星级评价
- **成绩记录**：本地存档，记录最佳成绩和历史数据
- **深色主题 UI**：现代化视觉设计，参考 type.fun 风格

## 初级模式 - 分阶段训练

### 学习路径

| 阶段 | 名称 | 按键范围 | 练习次数 |
|------|------|----------|----------|
| 1 | 基准键位 | A S D F J K L ; | 30 |
| 2 | 左手无名指 | 2 W S X + 基准 | 25 |
| 3 | 左手中指 | 3 E D C + 基准 | 25 |
| 4 | 左手食指上 | R T F V + 基准 | 25 |
| 5 | 左手食指下 | V G B + 基准 | 20 |
| 6 | 右手食指下 | H N M + 基准 | 20 |
| 7 | 右手食指上 | Y U J + 基准 | 25 |
| 8 | 右手中指 | I K , + 基准 | 25 |
| 9 | 右手无名指 | O L . + 基准 | 25 |
| 10 | 综合练习 | 全部字母 | 50 |

每个阶段都有最低准确率要求，未达标会重新练习当前阶段。

## 安装说明

### 系统要求

- macOS / Windows / Linux
- Python 3.8+
- Pygame 2.0+

### 安装步骤

```bash

git clone https://github.com/your-repo/keyboard-trainer.git
cd keyboard-trainer
```

## 技术亮点

### 1. 标准指法映射（已修正）

游戏使用标准美式QWERTY指法分配：

```
左手: 小指 ← 无名指 ← 中指 ← 食指 → 拇指
      1QA   2WS   3ED   4RFV  5TGB   (空格)
右手: 拇指 → 食指 → 中指 → 无名指 → 小指
      (空格) 6YHU  7JIK  8OL   9P;，
```

### 2. 指法合规追踪

`FingerComplianceTracker` 类提供：

- 每次按键的手指使用记录
- 指法合规率统计
- 手指使用均衡度评分
- 错误手指使用报告

### 3. 指法均衡算法

高级模式采用智能生成策略：

```python
# 优先选择使用次数最少的手指对应的字母
if random.random() < 0.8:
    selected_finger = min_usage_fingers[random]
    char = random.choice(finger_letter_map[selected_finger])
else:
    char = random.choice(alphabet)  # 20%随机保持多样性
```

## 常见问题

**Q: 为什么初级模式有10个阶段？**

A: 这是基于标准打字教学法的设计。从基准键位开始，逐步扩展到每个手指负责的键位，最后进行综合练习。每个阶段都有准确率要求，确保真正掌握后再进入下一阶段。

**Q: 指法均衡算法是如何工作的？**

A: 系统追踪每根手指的使用次数。生成下落字母时，80%概率选择当前使用最少的手指对应的字母，20%概率完全随机（保持游戏多样性）。

**Q: 如何使用姿势指导模块？**

A: 在初级模式开始前，会自动显示姿势指导页面。你也可以随时在游戏中按 H 键查看。
