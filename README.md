# 儿童键盘练习小游戏

这是一个专为11岁儿童设计的键盘练习小游戏，旨在帮助孩子熟悉电脑键盘布局、标准指法和基础中英文输入，提升兴趣、准确率和反应速度。

## 功能特点

- **四种难度模式**：
  - 初级模式：单键指法训练
  - 中级模式：英文短句打字
  - 高级模式：字母下落切水果玩法
  - 大师模式：汉字下落+拼音输入
- **全程音效和动画反馈**：每次按键都有即时视觉和音效反馈
- **鼓励系统**：提供积极的鼓励语言和星级评价
- **成绩记录**：保存本地成绩记录，显示每关的最佳成绩、最佳速度、纪录保持者
- **离线运行**：游戏完全离线运行，不依赖网络

## 安装说明

### 系统要求

- macOS
- Python 3.8 或更高版本
- Pygame 2.0 或更高版本

### 安装步骤

1. 确保已安装Python：
   ```bash
   python3 --version
   ```

2. 安装Pygame：
   ```bash
   pip3 install pygame
   ```

3. 下载或克隆本项目到本地：
   ```bash
   git clone https://your-repository-url/keyboard-trainer.git
   cd keyboard-trainer
   ```

## 使用说明

### 运行游戏

在项目目录下执行：
```bash
python3 main.py
```

或者使用启动脚本：
```bash
./start_game.sh
```

### 游戏操作

- **主菜单**：选择游戏模式、查看成绩档案或调整设置
- **初级模式**：根据屏幕提示按下对应按键
- **中级模式**：输入显示的英文单词、短语或句子
- **高级模式**：当字母下落到屏幕时，按下对应按键
- **大师模式**：输入下落汉字的拼音，按Enter确认
- **结束/返回**：按 ESC 或 End 立即结束本局并返回主菜单（不记录成绩）
- **暂停/继续**：仅高级/大师模式的暂停按钮可用，初/中级不提供暂停

### 游戏目标

- 初级模式：在100次按键中获得尽可能高的准确率和分数
- 中级模式：准确快速地输入显示的英文内容
- 高级模式：在时间结束前击中尽可能多的下落字母
- 大师模式：在时间结束前正确输入尽可能多的汉字拼音

## 游戏模式详解

### 初级模式

- 屏幕显示完整标准键盘
- 系统每次随机点亮一个按键
- 孩子按下对应键则判定成功并得分
- 每局固定100次按键挑战
- 记录正确次数、准确率、最终得分和星级评价

### 中级模式

- 训练内容从单个键升级为英文单词、短语、短句和短文
- 统计准确率、完成时间和速度（CPM/WPM）
- 界面中突出当前目标文本、已输入内容、剩余内容、错误高亮

### 高级模式

- 屏幕上有不断下落的英文字母
- 玩家在字母掉落过程中按下对应键，可将该字母"切碎"并得分
- 如果字母落到底部前没有被击中，则不得分
- 随着小局推进，字母下落速度逐渐增加

### 大师模式

- 掉落元素变成汉字
- 玩家需要输入该汉字的拼音，并按Enter确认
- 输入正确后，该汉字爆炸并得分
- 输入错误或超时则失分/漏判

## 技术说明

### 项目结构

```
keyboard_trainer/
├── main.py                  # 程序入口
├── config.py                # 游戏配置参数
├── scene_manager.py         # 场景管理
├── ui/                      # UI组件
├── game_modes/              # 游戏模式
├── keyboard/                # 键盘相关
├── content/                 # 游戏内容
├── effects/                 # 特效系统
├── audio/                   # 音频系统
├── data/                    # 数据存储
└── resources/               # 资源文件夹
```

### 自定义和扩展

- **添加新的词库**：编辑 `content/` 目录下的文件
- **修改游戏配置**：编辑 `config.py` 文件
- **添加新的关卡**：在 `game_modes/` 目录下创建新的模式类

## 注意事项

- 首次运行游戏时，会在 `data/` 目录下创建存档文件
- 游戏音效文件应放在 `resources/sounds/` 目录下
- 如果音效文件不存在，游戏仍可正常运行，但不会有音效
